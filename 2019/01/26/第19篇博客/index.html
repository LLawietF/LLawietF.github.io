第三章

1.C#提供16种预定义类型，包括13种简单类型和3种非简单类型

  • 简单类型

bool char decimal float double sbyte byte short unshort int uint long ulong

  • 非简单类型

string object dynamic

2.有6种类型可由用户自己创建

class struct array enum delegate interface

3.值类型数据存放在栈中，引用类型引用存放在栈，数据存放在堆中；引用类型的所有数据成员都存放在堆中，无论它是值类型还是引用类型。

变量类型 存储位置 自动初始化
本地变量 栈或者栈和堆 否
类字段 堆 是
结构字段 栈或堆 是
参数 栈 否
数组 堆 是

1.类字段每种类型的默认值是0，bool型为false，引用类型为null。

第五章

1.var

  • 只能用于本地变量，不能用于字段
  • 只能在变量声明并且初始化时使用
  • 一旦编译器推断出变量的类型，它不能更改

2.值参数

  • 通过将实参的值复制到形参的方式把数据传递给方法
  • 方法被调用时，系统做如下操作：
  1. 在栈中为形参分配空间
  2. 将实参的值复制给形参

3.引用参数

  • 使用引用参数时，必须在方法的声明和调用中都使用ref修饰符
  • 实参必须是变量，在用作实参前必须被赋值

4.输出参数

输出参数有以下要求：

必须在声明和调用中都使用修饰符out；
实参必须是变量；
在方法内部，输出参数在能够被读取前必须被赋值，没有必要再方法调用之前为实参赋值；
在方法返回之前，方法内部贯穿的任何可能路径都必须为所有输出参数进行一次赋值。

5.可变参数

参数数组的注意点：

  • 在一个参数列表中只能有一个可变参数

  • 必须是列表中最后一个

  • 所有参数都必须具有相同的类型

可以使用两种方式为参数数组提供实参：

  • 一个逗号分隔的该数据类型元素的列表
  • 一个该数据类型的一维数组

6.可选参数

可选参数注意点：

  • 只要值类型的默认值在编译时可以确定，就可以使用值类型作为可选参数

  • 只有在默认值是null的时候，引用类型才可以作为可选参数来使用

  • 所有必填参数必须在可选参数之前声明，如果有params参数，必须在所有可选参数之后声明

  • 必须从可选参数列表的最后开始省略，一直到列表开头
